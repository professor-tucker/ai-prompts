Windows Deployment Instructions

1. Install Docker Desktop

- Download and install Docker Desktop from the official Docker website.

- Follow the installation instructions and enable the WSL 2 feature.

2. Clone the Repository

- Open PowerShell or Command Prompt and run:


git clone <repository_url>
cd <repository_directory>

3. Build and Start the Services


docker-compose up --build

4. Access the Services

- Flask Application: http://localhost:5000

- Grafana: http://localhost:3000

- InfluxDB: http://localhost:8086

Additional Notes and Documentation

Prerequisites

Ensure you have git installed on your system. You can download and install Git from the official Git website.

Starting Docker Desktop

Ensure Docker Desktop is running before executing Docker commands.

Stopping the Services

Press Ctrl+C in the terminal where Docker Compose is running, and then run:


docker-compose down
